# MIDI Piano Player

This React app plays uploaded MIDI files with a piano-style UI. Notes light up on the 88-key keyboard while the transport runs.

## Features

- Upload .mid or .midi files
- Play/stop controls with tempo slider
- Visual keyboard highlighting during playback

## Getting Started

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal and upload a MIDI file.

## Tech Stack

- React + TypeScript
- Vite
- Tone.js + @tonejs/midi
